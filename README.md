# 黄金迷宫

一个基于 Vite + Vue 3 + TypeScript + Three.js 的 3D 迷宫游戏。场景风格为明亮温暖的金黄色沙漠遗迹，包含可重复生成迷宫、角色背后跟随视角、键盘移动、碰撞检测、出口胜利判定和完整 UI。

## 启动

```bash
pnpm install
pnpm run dev
```

如果 Windows 环境提示 pnpm junction / untrusted mount point，可临时使用：

```powershell
$env:PNPM_CONFIG_NODE_LINKER="hoisted"; pnpm install
$env:PNPM_CONFIG_NODE_LINKER="hoisted"; pnpm run dev
```

如需重新生成本地贴图资源：

```bash
pnpm run generate:textures
```

生成的贴图位于 `public/assets/textures/`。

## 操作

- `W` / `↑`：前进
- `S` / `↓`：后退
- `A` / `←`：向左转身移动
- `D` / `→`：向右转身移动
- `Shift`：奔跑
- `M`：显示 / 隐藏辅助路线
- `N`：显示 / 隐藏迷宫俯瞰图
- `R`：重置到当前迷宫起点
- `Esc`：释放鼠标并暂停
- 鼠标左键按住水平拖动：调整左右视角，角色朝向同步旋转

## 功能

- 默认角色背后跟随相机，点击开始后启用 Pointer Lock，按住鼠标左键只旋转左右视角。
- Q 版探险家角色由 low-poly Three.js 几何体组合，支持待机、行走、奔跑动作。
- DFS 回溯生成 `S / E / 0 / 1` 二维数组迷宫，并保证起点到终点可达。
- 简单、普通、困难三档难度，对应不同地图尺寸和开孔程度。
- 网格碰撞检测，按 X/Z 分轴移动以支持贴墙滑动。
- 连续墙段渲染砂岩墙体，减少格子墙之间的拼接感，并支持阴影、雾效、暖色天空和终点光效。
- 辅助路线使用更醒目的发光粗线和动画箭头，俯瞰图可显示角色和出口位置。
- HUD、开始界面、暂停界面和胜利弹窗均为金色遗迹主题。
- 资源文件缺失时，运行时会生成 Canvas 兜底纹理，确保项目仍可运行。

## 微信小游戏版

小游戏项目位于 `mini-game/`，PC Web 版代码保持在项目根目录，两端互不覆盖。

导入微信开发者工具：

```bash
cd mini-game
npm install
```

然后在微信开发者工具中选择“导入项目”，项目目录选择 `D:\github\maze-game\mini-game`，AppID 可先使用测试号。导入后执行“工具 -> 构建 npm”，并切换到横屏模拟器预览。

移动端操作：

- 左下角摇杆：控制角色移动，轻推慢走，推到底奔跑。
- 右半屏左右滑动：只控制水平视角旋转，无上下俯仰和惯性。
- 右上角：暂停 / 继续 / 胜利后再来。
- 右侧按钮：地图、提示路线、自动寻路 / 停止寻路。
- 右下角：相机回正。

小游戏端保留第三人称角色视角、随机迷宫、碰撞、出口胜利、俯瞰图、路线提示和自动寻路。真机上仍建议重点确认横屏安全区、触摸手感、低端机性能和 Three.js 在微信运行时的兼容性。
